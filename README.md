## Getting Started for App

1. run pod install
```console
$ pod install
```

2.  open RxSwiftMVVM.xcworkspace
```console
$ open RakutenTV.xcworkspace
```

3. run  RakutenTV

## Getting Started for UnitTest or UITest

1. run pod install
```console
$ pod install
```

2.  open RxSwiftMVVM.xcworkspace
```console
$ open RakutenTV.xcworkspace
```
3. run  RakutenTVTest or RakutenTVUITest

4. run  RakutenTV
